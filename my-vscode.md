# 自己配置 快捷键想法
## 统一规则
> 最舒服的两个按键就是 command 和 control。所以尽量围绕着两个键
### 窗口区
- command + shift + f : 打开工作区
- command + shift + n : 打开新窗口
### 功能区
- command 负责编辑功能区
  - command + 上下左右：移动功能区 
  - command + \ : 创建左右编辑功能区
  - command + | : 创建上下编辑功能区
- control 负责编辑窗口以外的窗口开合
  - control + f 左侧搜索菜单
  - control + ; 左侧文件菜单
  - control + g 左侧git菜单
  - control + c 下侧终端
### 编辑区
- command + [] 缩减
- alt + [] 闭合 (因为control + [ 被vim占用了 - -)
