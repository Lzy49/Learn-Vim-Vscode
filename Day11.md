# 多文件操作
## 标记 跳转
### 标记
利用 m 键标记：
- m + 任意小写字母 同文件标记
- m + 任意大写字母 多文件标记。

### 跳转
- `'` + 对应字母 跳转到标记行
- `/`` + 对应字母 跳转到标记为止
## 函数查询工具
命令 : gd
### 使用方法
> 变量和函数是同样的功能。同样的操作
- 在函数调用的地方按，会移动到函数声明的位置（跨文件）
- 在函数声明的地方按，会出现查询菜单
  - j 下
  - k 上
  - h 返回到文件名
  - l 展示文件中包含的位置
  - esc 取消
  - 在想要进去的位置 按l 进入
## 移动命令
-  { 和 } 移动代码块（空行区分）
## 移动历史
- 查看移动历史记录 :jumps 
- Control + i 向后跳
- Control + o 向前跳
