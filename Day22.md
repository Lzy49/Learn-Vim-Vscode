# 编码ing
## 提示
- `command + .` 展示 vscode 推测的行为,错误或警告的解决
- `command + shift + space` 展示 vscode 参数提示
- `command + i` 展示 vscode 变量函数推导。
## 行处理
- `option + up/down` 移动行
- `command + enter` 增加行
- `command + shift + enter` 增加上行
- `option + delete` 删除前面的单词（连接的就是一个单词）
- `option + ctrl + delete` 删除当前的单词（驼峰）
- `f8` 跳转到错误处（全局）
- `command + f2` 选中当前单词（多个）
