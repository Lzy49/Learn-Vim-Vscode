# 宏是什么
宏就像ps中的记录存储成一个快捷操作一样。一般用在重复某个操作。
# 宏技巧
- 编辑宏的第一步应该是将光标规定好位置。这样方便在任何地方使用
- 移动时使用相对位置例如 w,e,f 这种。 不要使用 hjkl 因为格式会很容不同
- 重复可以写一个超过实际的大小，当执行到位置后会自动停止。
# 宏命令
- q + 小写字母: 增加或替换宏 开始
- q : 完成宏 结束记录
- :reg + 小姐字母： 查看宏。
- @ + 小写字母： 使用宏
- @@ : 使用上一个宏
- number + @@ ： 使用 number 次宏
- q + 大写字母： 追加宏
