# 替换字符串
## substitute
- 开启公式：/ 是必要分隔符。 ` : + 范围 + s / 要替换的内容（ 正则 / 字符串)  / 替换文字 / 匹配模式 ` 
- 范围
  - `$` : 到尾部
  - `%` : 全文
  - `number,number` : 几行到几行
- 匹配模式
  - `g` 全文替换
  - `c` 弹出对话框来一个一个去选择
    - `a` : 全部替换
    - `n` : 下一个
    - `y` : 替换并下一个
    - `l` : 替换并退出
    - `esc` : 退出
- 在可视化模式进入 substitute 模式会将选中文字其中匹配内容全部替换。
# gb
- 在想要选中单词 按 gb 选中下一个。
- gb 选择是区分大小写的。
