# vscode 搜索
## 全局搜索
- `command + shift + f` 打开搜索窗
  - `command + 下` 移动到搜索搜索结果中
  - `hjkl` 移动，打开，文件，文件夹
  - `shift + 方向` 移动回结果 
## 搜索特定内容
- `command + t` 【#】 全工作区搜索函数，变量,关键字
- `command + O` 【@】当前文件搜索 函数， 变量 ， 关键字
  - `命令 + :` 排序
- `command + P` 搜索软件命令（包括插件）
- `command + p` 搜索文件。
# 切换文件
- `control + tab` 切换当前打开的文件
